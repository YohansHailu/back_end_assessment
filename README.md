# Task Manager Web Application


## Features
### User Registration
- New users can easily register using their email and password.
### Task Creation and Management
- users have the ability to create, update, view, and delete tasks.
- Each task comes with essential details: title, description, due date, and status (e.g., "To Do," "In Progress," "Done").
- Users can mark tasks as complete, helping to track progress effectively.

### Project and User Assignment
- Users can create projects, enabling better organization of related tasks.
- Tasks can be assigned to specific projects, streamlining project management.
- Tasks within projects can be assigned to specific users, promoting collaboration and accountability.
### Task Priority and Labels
- Users can assign priority levels to tasks, aiding in task prioritization (e.g., low, medium, high).
- Labels can be added to tasks, allowing users to categorize tasks based on context (e.g., "Work," "Personal").

#setup instructions
To get started with the Task Manager web application, follow these steps:
1. Clone the repository to your local machine.
2. Open the project in your preferred ASP.NET development environment.
3. Configure the database connection settings to ensure proper data storage.
4. Build and run the application.
5. Access the application through your web browser and start managing your tasks and projects.
